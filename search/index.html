Simple search

The search is based on Elasticsearch and the regular Elasticsearch search API is exposed at

http://…/api/v1/search

Example requests

  • http://o2r.uni-muenster.de/api/v1/search?q=*
  • http://o2r.uni-muenster.de/api/v1/search?q=term
  • http://o2r.uni-muenster.de/api/v1/search?q=*&fields=compendium_id

Security

Only reading requsts, i.e. GET requests, are allowed at the moment. This means that Kibana cannot be run against this interface, and neither can complex queries requiring POST requests. GET requests with payloads are allowed.

Example query:

curl -XGET 'https://…/api/v1/search' -d '{
    "query": {
            "type" : {
                "value" : "jobs"
            }
        }
    }
}'
